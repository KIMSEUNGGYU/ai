## 곽호영 리뷰 시뮬레이션

### 예상 반응
💬 이거 보고가 아니라 일기장이네... 뭘 판단하라는 건지 모르겠음
❓ "몇 군데요? 어디어디에요?"
❓ "반응이 괜찮았다는게 구체적으로 뭐가 괜찮았어요? 계약 의향이 있었어요? 수수료 조건이 맞았어요?"
❓ "추가로 더 진행한다는게 언제까지 몇 군데 더 하는거에요?"
❓ "우리 협력대리점 현재 몇 개이고 이번달 목표가 몇 개인데?"

### 판정
❌ 반려 — 숫자 전무, 비교 기준 없음, 구체적 결과 없음, 실행 일정 없음. 보고에 들어가야 할 요소가 하나도 없다.

### 보완 포인트
- [ ] 미팅한 대리점 수, 업체명, 각 업체 규모(예상 가맹점 수 등)
- [ ] "반응이 괜찮았다"의 구체적 근거 (계약 의향, 조건 협의 내용 등)
- [ ] 현재 협력대리점 현황 수치 (기존 몇 개, 월별 추이)
- [ ] 추가 진행 계획: 대상 업체, 미팅 일정, 결과 공유 예정일
- [ ] 목표 대비 현재 진척률

### 개선 예시
> 협력대리점 확대 건입니다. 이번 주 3곳(A대리점·B대리점·C대리점) 미팅 다녀왔습니다.
> - A대리점: 월 설치 30건 규모, 수수료 조건 협의 완료 → 계약 의향 있음
> - B대리점: 월 설치 20건 규모, 추가 미팅 필요 (3/13 예정)
> - C대리점: 월 설치 15건 규모, 긍정적이나 내부 검토 중 (3/17까지 회신)
> 현재 협력대리점 12개(월 설치 합산 280건), 이번달 목표 15개(350건)입니다. 추가 2곳 더 미팅 예정(3/18까지)이고, 결과는 3/20까지 정리해서 공유드리겠습니다.
