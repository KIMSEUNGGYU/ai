# HEARTBEAT.md

## 1주차 루틴 (시험 운영)

1) 오전 체크 (09:00~11:00)
- 오늘 일정/핵심 할 일 3개 브리프

2) 오후 체크 (14:00~16:00)
- 진행 점검, 막힌 이슈 1~2개 정리

3) 야간 마감 (21:00~23:00)
- `memory/YYYY-MM-DD.md`에 오늘 결정/다음 액션 업데이트

4) 2~3일마다
- 최근 daily 로그를 보고 `MEMORY.md` 승격 후보 1~3개 반영

## 규칙
- 새 내용 없으면 `HEARTBEAT_OK`
- 늦은 밤(23:00~08:00)에는 긴급 건만 알림
