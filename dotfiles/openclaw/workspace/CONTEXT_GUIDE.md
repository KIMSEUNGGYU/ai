# CONTEXT_GUIDE.md

## 목적
AI가 긴 대화 없이도 정확히 일하도록 컨텍스트를 최소 단위로 공급한다.

## 주입 순서 (항상 동일)
1. TASK.md (현재 목표/완료조건)
2. STATE.md (최근 진행/다음 액션)
3. PROJECT.md (고정 규칙)
4. 필요한 코드/로그 발췌 (10~30줄)
5. DECISIONS.md 관련 항목 1~2개

## 1회 요청 포맷 (복붙용)
- 목표:
- 완료조건:
- 관련 파일(최대 3개):
- 코드/로그 발췌(10~30줄):
- 제약:
- 이번 턴 해야 할 일 1개:

## 운영 규칙
- 한 턴 한 이슈.
- 전체 파일 대신 필요한 줄만.
- 5~10턴마다 STATE.md 갱신.
- 결정 나면 DECISIONS.md 즉시 기록.
- 실패 해결하면 RUNBOOK.md 즉시 기록.
