# 발표자료 생성

## 입력
- `$ARGUMENTS`: 보고서 파일 경로 (예: `보고서/주간트렌드보고서_W1.md`)

## 작업 절차

1. **보고서 읽기**: `$ARGUMENTS` 경로의 보고서 파일을 읽는다
2. **핵심 내용 추출**: 보고서에서 슬라이드에 들어갈 핵심 내용을 추출한다
3. **pptx 스킬 활용**: `document-skills:pptx` 스킬을 사용하여 PPTX를 생성한다
4. **저장**: `발표자료/` 폴더에 저장

## 슬라이드 구성 (6-8장)

1. **표지**: 보고서 제목, 작성일, 작성팀
2. **핵심 트렌드 3줄 요약**: 이번 주 핵심 메시지
3. **주요 뉴스 TOP 3**: 각 뉴스의 핵심 + 시사점
4. **기술 트렌드**: 기술별 성숙도 표
5. **시장 트렌드**: 투자·시장 변화 핵심
6. **경쟁사 동향**: 주요 기업 전략 요약
7. **액션 아이템**: 우선순위별 대응 방안
8. **다음 주 관전 포인트**: 주목할 이벤트

## 디자인 원칙
- 슬라이드당 핵심 메시지 1개
- 텍스트 최소화, 키워드 중심
- 표와 불릿포인트 활용
- 경영진이 30초 내에 각 슬라이드를 파악할 수 있도록

## 파일명 규칙
- 보고서 파일명에서 주차 추출
- `발표자료/주간트렌드_W{주차}.pptx`로 저장
