  "기록해줘", "배운 거 정리해줘", "학습 정리", "TIL 작성", "오늘 배운 것 정리",
  "인사이트 정리해줘", "노트 남겨줘", "이거 기억해둬", "나중에 볼 수 있게 정리" 등의 표현에 트리거.
  대화 중 학습 내용이 있고 사용자가 저장 의도를 보이면 반드시 트리거한다.
  단순 대화 요약이 아닌, "몰랐던 것 → 알게 된 것"을 추출하여 복습 가능한 형태로 저장.
  /doc은 외부 자료 포함 구조화 문서용, /note는 대화 학습 추출에 집중.
---

사용자가 세션 대화 중 정리/저장을 요청할 때 트리거된다.

## 동작

`/note` 커맨드와 동일한 로직을 수행한다.

1. 학습 내용 추출 (몰랐던 것 → 알게 된 것)
2. 저장 위치 자동 판단 + 사용자 확인
3. 기존 문서 확인 (있으면 업데이트)
4. 복습 가능한 형태로 저장

상세 동작은 `/note` 커맨드 참조.
